## 📁 About This Project

This portfolio was built from scratch as part of my Full Stack Development internship. It follows a strict set of rules — no frameworks, no libraries, no shortcuts. Every section is structured using pure semantic HTML5 and every style is written in an external CSS3 file.

The goal was simple: prove that you don't need Bootstrap, React, or a UI library to build something that looks and feels professional.
